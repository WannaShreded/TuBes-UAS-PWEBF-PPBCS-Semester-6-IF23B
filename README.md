# HNRZ - Dashboard Admin
Aplikasi dashboard admin berbasis Laravel 13
untuk mengelola admin

## Fitur Utama
- Autentikasi & manajemen role-permission
- Dashboard statistik + grafik interaktif
- CRUD: [Entitas 1], [Entitas 2], [Entitas 3]
- Search, sort, filter, pagination

## Tech Stack
Laravel 13 • PHP 8.3 • MySQL 8 • Tailwind CSS

## Instalasi
git clone ... && cd repo
composer install && npm run build
php artisan migrate --seed
php artisan serve
